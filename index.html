Research Projects

Major Research and Development Projects

Vision-Language Model (VLM) for Desktop AI

Developed an optimized Vision-Language Model for on-device AI applications, enabling seamless multimodal interactions on desktop platforms.

Deep Learning VLM On-Device AI

Light Field Display Research

Designed localized 2D-to-3D conversion algorithms for stereoscopic rendering, developed cross-talk reduction algorithms based on Light Field image processing, built deep learning-based multi-layer Light Field decomposition models, and developed real-time telepresence systems for large screens.

Light Field 3D Display Deep Learning

AI for Image Quality Enhancement

Enhanced MicroLED image quality through subpixel rendering using DNN, developed super-resolution models for upscaling SD/HD/FHD to 4K/8K on digital TVs, and developed single-image HDR enhancement technology for smartphone photography.

Super-Resolution HDR Image Processing

VR Motion Sickness Mitigation

Developed a system that analyzes VR content motion sickness severity in real-time using image processing techniques, dynamically adjusts field of view, and stabilizes video to reduce user discomfort.

VR Computer Vision Real-time Processing

Video Coding Standardization (VVC)

Contributed to the development of next-generation video compression algorithms after HEVC, improving coding efficiency.

Video Coding VVC Standardization
