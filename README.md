# Kyungmin Lim - Personal Website

A modern, responsive personal website showcasing research projects, publications, and professional information.

## Overview

This is the personal website of Kyungmin Lim, Principal Engineer at Samsung Research. The website features a clean, modern design with smooth animations and full mobile responsiveness.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean interface with gradient effects and smooth transitions
- **PWA Support**: Progressive Web App with manifest.json for mobile installation
- **Social Integration**: Links to GitHub, Google Scholar, and LinkedIn
- **Project Showcase**: Visual presentation of research projects with images
- **Publications & Awards**: Comprehensive list of academic achievements

## Sections

1. **Home**: Hero section with introduction and social links
2. **About**: Career, education, and technical skills
3. **Projects**: Research projects with visual cards
   - Vision-Language Model (VLM) for Desktop AI
   - Light Field Display Research
   - AI for Image Quality Enhancement
   - VR Motion Sickness Mitigation
   - Video Coding Standardization (VVC)
4. **Publications**: Research papers, awards, and US patents
5. **Contact**: Email contact information

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables, flexbox, and grid
- **JavaScript**: Smooth scrolling, mobile menu, and interactive features
- **Font Awesome**: Icon library for social media icons
- **Academicons**: Academic icon library for Google Scholar

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Kyungmin-Lim/kyungmin-lim.github.io.git
   ```

2. Open `index.html` in a web browser or serve using a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. Access the website at `http://localhost:8000`
